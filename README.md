# FoodTrip

FoodTrip helps people decide where to eat fast, based on where they are right now.

## Problem It Solves

Choosing food nearby usually means jumping between multiple apps, guessing distance, and checking places one-by-one.

FoodTrip solves this by giving a single map-first view of nearby food spots with practical decision signals:

- Distance and ETA by travel mode
- Open/closed status and ratings
- Search, recent lookups, favorites, and bookmarks
- Unified place results from multiple providers

The goal is simple: reduce decision time from "Where should we eat?" to "Let's go here."
